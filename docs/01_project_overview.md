# Project Overview

# E-commerce Analytics ETL Pipeline

## Project Summary

This project demonstrates how a Data Engineer designs and implements an end-to-end ETL (Extract, Transform, Load) pipeline using a real-world public e-commerce dataset. The pipeline extracts raw transactional data, transforms it into analytics-ready datasets, and loads the cleaned data into a relational database for business reporting.

The project simulates a real business environment where executives and business teams require reliable data for decision-making.


## Objectives

* Build a complete ETL pipeline using Python, SQL, and MySQL.
* Clean and transform raw e-commerce data into business-ready datasets.
* Design a relational database schema for analytical workloads.
* Produce SQL queries that answer common business questions.
* Demonstrate practical Data Engineering skills suitable for an entry-level Data Engineer role.


## Dataset

Dataset: Brazilian E-Commerce Public Dataset by Olist

Tables used:

* Customers
* Orders
* Order Items
* Products


## Business Goals

The transformed dataset supports business users in answering questions such as:

* How much revenue is generated over time?
* Which products contribute the most revenue?
* How long does delivery take on average?
* How many customers return for multiple purchases?
